# script_Powershell
A script that allows you to select a work or rest mode on your computer

How to make it work:

Launch PowerShell in administrator mode.

Navigate to the directory where you downloaded the script by using cd or Set-Location (sl).

Launch it by running .\script.ps1.

![Script_enable](https://github.com/user-attachments/assets/670ea2b6-a684-4f2a-b111-20bc5a0564d4)

 If it's not working, try this command in your PowerShell:

Set-ExecutionPolicy RemoteSigned
