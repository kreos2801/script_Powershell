# script_Powershell
A script that allows you to select a work or rest mode on your computer

how to make it work
Lauch powershell in administrator mode
Go where you download the script by using cd or SL(set-location)
lauch it by doing .\script.ps1
![Script_enable](https://github.com/user-attachments/assets/670ea2b6-a684-4f2a-b111-20bc5a0564d4)

 If it's not working try this command in your powershell
 Set-ExecutionPolicy RemoteSigned
